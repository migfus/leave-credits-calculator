## About The Project

[![Product Name Screen Shot][product-screenshot]](https://migfus.site/images/lcc-app/app-preview.png)

Managing employee leave shouldn't feel like a manual data-entry chore. While many HR systems exist, most are either overly complex or lack a straightforward way to handle the granular logic of credit accruals. I built this Leave Credit Calculator to bridge that gapâ€”providing a fast, reliable, and developer-friendly solution to a universal HR headache.

Here's why this tool stands out:

- Time is your most valuable asset: HR professionals shouldn't spend hours cross-referencing spreadsheets and manual logs. Your focus should be on people, not formulas.
- Precision by default: Manual calculations are prone to human error. This tool automates the math to ensure every minute of earned leave is accounted for accurately.
- Developer-Centric Design: Built with a clean stack and strict naming conventions, the codebase is as organized as the data it produces.
- Simplicity is Sophistication: It strips away the bloat of enterprise software, offering a lean interface that does exactly what you needâ€”nothing more, nothing less.

No more re-calculating balances from scratch every month. Whether you're a small team or scaling up, this tool is designed to make leave management a "set it and forget it" part of your workflow.

### Installation

_Install the following repo by following the instruction. If there's any issue upon installation please contact my email: migfus20@gmail.com_

1. Clone the repo
   ```sh
   git clone https://github.com/migfus/leave-balance-calculator.git
   ```
2. Install NPM packages
   ```sh
   npx expo install
   ```
3. Make sure it passed on expo-doctor, if any needed to update just follow the expo doctor.
   ```sh
   npx expo-doctor
   ```
4. Run the app
   ```sh
   npx expo run start
   ```
5. Change git remote url to avoid accidental pushes to base project
   ```sh
   git remote set-url origin github_username/repo_name
   git remote -v # confirm the changes
   ```
